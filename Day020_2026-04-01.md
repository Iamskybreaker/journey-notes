## 🧠 What I Learned

- **MAC Address:**  
  • A unique hardware identifier assigned to a device’s network interface  
  • Used for communication within a Local Area Network (LAN)  

- **Role in Networks:**  
  • Enables devices to identify and communicate with each other on the same network  
  • Switches use MAC addresses to forward data to the correct device  

- **MAC vs IP Address:**  
  • MAC Address → physical identifier (local communication)  
  • IP Address → logical identifier (used across networks)  
  • MAC handles delivery within LAN, IP handles routing between networks
