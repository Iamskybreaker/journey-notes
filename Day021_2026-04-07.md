## 🧠 What I Learned

- **DNS (Domain Name System):**  
  • Translates human-readable domain names into IP addresses  
  • Acts as a bridge between users and servers on the internet  

- **Why DNS is Necessary:**  
  • Humans remember domain names more easily than IP addresses  
  • Enables seamless access to websites without needing to know their IP  

- **DNS Resolution Process:**  
  • User enters a domain name  
  • Request is sent through a series of DNS servers  
  • The correct IP address is returned  
  • The browser uses the IP to communicate with the destination server
